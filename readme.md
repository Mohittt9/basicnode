## This is a simple Node.js application with EJS templating engine.
## This project teach us how to use node js for render ejs templates.
 
 ## Installation
1. Clone the repository.
2. Install dependencies: `npm install`
3. Install nodemon: `npm install -g nodemon`
4. Install ejs: `npm install ejs`


## Usage
1. Start the server: `npm start`
2. Open your browser and navigate to `http://localhost:3000`    
3. You can visit the following routes:
   - Home: `/`
   - About: `/about`
   - Contact: `/contact`
## Technologies Used
- Node.js

- ejs
